# KFC订单系统
包含客户端的登录和注册页面、订单模块页面、我的模块页面、购物车页面；
该系统有搜索菜品功能，查看菜品详情功能，查看订单功能、订购商品的功能。

## 技术栈
Vue、Axios、VueRouter、VantUI组件
